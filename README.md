## 🚀 Cómo Usar

### Opción 1: Abrir directamente
1. Abre el archivo `index.html` en tu navegador web
2. ¡Listo! La página funcionará completamente

### Opción 2: Con servidor local (recomendado para desarrollo)

#### Usando Python:
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

#### Usando PHP:
```bash
php -S localhost:8000
```

#### Usando Node.js (con live-server):
```bash
npx live-server
```

Luego abre tu navegador en: `http://localhost:8000`

## 📝 Personalización

### Cambiar Imágenes
Las imágenes actuales son de Unsplash (placeholder). Para usar tus propias imágenes:

1. Guarda tus imágenes en una carpeta `images/`
2. Reemplaza las URLs en `index.html`:
```html
<!-- Ejemplo actual -->
<img src="https://images.unsplash.com/photo-..." alt="...">

<!-- Cambiar por -->
<img src="images/tu-imagen.jpg" alt="...">
```

### Cambiar Información de Contacto
Busca y reemplaza en `index.html`:
- Teléfono: `947 469 343`
- Dirección: `X53Q+68C, Sta. Rita, Caraz 02167`

### Modificar Colores
Edita las variables CSS en `styles.css`:
```css
:root {
    --color-primary: #0891b2; /* Cambia este valor */
    --color-secondary: #f59e0b; /* Y este */
    /* ... más colores ... */
}
```
